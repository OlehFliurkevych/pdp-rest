# pdp-rest

project for resolving tasks from _**Rest API**_ module in scope of preparing to assessment.
***
### Spring-based module, which handle CRUD operations with event.

Technologies:
- JDK 11+
- Apache Maven 3.6.0+
- Git
- Docker
- OpenApi
- Postgres
***
Build and deploy instance of Postgres DB in Docker:
> docker-compose up

Generate JavaDoc:
> mvn javadoc:javadoc
